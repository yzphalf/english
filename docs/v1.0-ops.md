# Stable v1.0 Ops Guide

Domain: `english.051231.xyz`

External URL (fixed): `https://english.051231.xyz:8443`

## 1) One-click install (new VPS)

```bash
bash /opt/english/app/scripts/v1_install.sh
```

If repo is not yet cloned, run from any location with overrides:

```bash
curl -fsSL https://raw.githubusercontent.com/yzphalf/english/main/scripts/v1_install.sh -o /tmp/v1_install.sh
chmod +x /tmp/v1_install.sh
DOMAIN=english.051231.xyz REPO_URL=https://github.com/yzphalf/english.git bash /tmp/v1_install.sh
```

## 2) One-click release (daily update)

```bash
bash /opt/english/app/scripts/v1_release.sh
```

## 3) One-click recovery (keep topics/comments)

```bash
bash /opt/english/app/scripts/recover.sh
```

## 4) One-click check

```bash
bash /opt/english/app/scripts/v1_check.sh
```

## Cloudflare notes

- DNS record: `english -> VPS_IP` (proxied/orange cloud)
- SSL mode: `Full` (or `Full (strict)` if origin cert chain is clean)
- Student entry URL for class: `https://english.051231.xyz:8443/student`

## Important

- Keep `/opt/english/app/data/app.db*` files.
- Do not delete `data/` on deploy.
